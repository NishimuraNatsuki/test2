# 走らないメロス
## 概要
### 「走れメロス」を形態素解析とマルコフ連鎖を用いて、物語をめちゃくちゃにしています。
### 入眠用の読み聞かせとしておすすめです。

## 使い方
### playで文章生成、resetでリセット、voiceで全ての文章を読み上げてくれます。

## 使用した言語、ライブラリ
### 言語はPythonです。
### flaskを使用して作成しています。
### Mecab,markovify,voicevoxなど様々なライブラリを使用しています。必要なライブラリは適宜ダウンロードするようお願いします。
